# TOBI

**Your playlists. Your vibe. Your story.**

TOBI turns your Spotify listening data into shareable cards and brutally honest AI recaps. Discover your "mouthly" top tracks and artists with a fresh, futuristic aesthetic.

## Features

- **Spotify Integration**: Connect your account to fetch top tracks and artists.
- **AI Recaps**: Generate brutal & personalized summaries of your listening habits.
- **Shareable Cards**:
  - **Dev Card**: Code-inspired aesthetic.
  - **Receipt Card**: Classic receipt style.
  - **Portfolio Card**: Bento grid layout.
  - more soon...

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer_motion.dev/)
- **Auth & Database**: [Supabase](https://supabase.com/)

## Getting Started

1.  **Clone the repo**
2.  **Install dependencies**:
    ```bash
    pnpm install
    ```
3.  **Run the development server**:
    ```bash
    pnpm run dev
    ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
